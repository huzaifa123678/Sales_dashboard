This is an interactive **Sales Dashboard** built with **Streamlit**, **Pandas**, **Matplotlib**, and **Seaborn**.  
It uses a cleaned sales dataset (`cleaned_sales_data.csv`) and provides insights into sales performance.

